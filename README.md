
# OpenGL / Carga 3DS

Sencilla aplicación para renderizar archivos 3DS.
Desarrollada en C++, con las librerías de OpenGL sobre Windows como Sistema Operativo y Dev-C++ como IDE.

## Uso

Ejecutar el archivo Leer3DS.exe, por defecto carga el archivo mapa.3DS, si se quiere visualizar otro archivo 3DS se tendrá que renombrarlo como mapa.3DS

Controles:

- Flechas: Rotación de la cámara
- RePag: Acercamiento de la cámara
- AvPag: Alejamiento de la cámara
- A: Renderizado de la rejilla de los objetos
- S: Renderizado sólido de los objetos
